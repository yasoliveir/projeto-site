# projeto-site
Projeto de um site criado durante o curso de Git e GitHub do curso em video

# Itens Assist. Liter.
- Histórico de Leitura (o mais importante)
  - Para cada livro já lido:
    - Título
    - Gênero(s)
    - Nota pessoal (ex: 1 a 5 estrelas)
    - Gostou? Por quê? (campo aberto ou tags)
    - Emoção causada (ex: empolgante, triste, reflexivo)

  - Preferências Literárias (como você gosta de ler)
    - Gêneros favoritos (com peso: amo / gosto / evito)
    - Subgêneros específicos (Ex: fantasia épica, sci-fi distópica, romance histórico)
    - Temas preferidos (Amizade, política, saúde mental, magia, guerra, amadurecimento etc.)
    - Temas que evita (Violência extrema, abuso, finais tristes, religião, etc.)
    - Ritmo rápido ou lento
    - Mais ação ou mais introspecção
  
  - Emoções e Experiência Desejada
      - Como você quer se sentir ao ler? (Confortado, Desafiado, Em choque, Inspirado)
      - Tipo de final preferido (Feliz, Agridoce, Aberto)
      - Livros para momentos específicos (Ler antes de dormir, Para sair de uma ressaca literária, Para aprender algo novo)
      - Comparações: (“Quero algo no estilo de X, mas mais Y”)
